# 02 · 游戏机制详解

> 来源: 游民星空 / TapTap / 3DM / 腾讯官方公告 / 百度百科 / 灰机 wiki. 所有机制以 2026-04-21 为时点; 后续版本可能调整.

---

## 1. 异色保底机制 (核心)

### 1.1 触发条件
- 玩家在**同家族/同系别**池中连续捕捉精灵, 累计足够的捕捉会在该区域召唤**梦魇污染体** (紫光, 固定 4 技能, 天赋"了不起", 有一技能为"奇异技能").
- 官方描述: **每捕捉 20 只同家族精灵** → 触发 1 次污染体出现.

### 1.2 保底规则
- 每次**击破污染护盾**: 约 **1.8%** 概率该污染即为异色形态 (官方补偿后**综合概率 ≈ 2%**).
- **硬保底**: 同家族池累计**击破 80 次** → 第 80 次**必出异色**.
- **击破护盾即计数**, 无需成功捕捉.
- 不同家族池**计数独立**; 跨家族不重置, 只是不推进.

### 1.3 "1.8% vs 2%" 的隐含信息
官方声明 "基础 1.8% / 补偿后综合 2%" — 如果只有硬保底, 则
$$
p_{\text{综合}} = 1 - (1-p)^{80} / 80 \approx \text{far above 2%}
$$
即在硬保底下平均需要约 42 次出货, 综合概率自然高过 2%.
因此 **"2%"** 暗示着**存在某种概率调整 / 软保底 / 早期保护机制** — 这是本研究要辨识的未知量.

### 1.4 攻略圈民间玄学
- "70-75 次冲刺大法": 攻略主张到 70-75 次时集中刷, 防止保底前掉线或切池.
- "3x3 速刷法": 在地图上 3 个刷新点轮转, 最小化移动耗时.

## 2. 精灵回顾 (返利) 系统

### 2.1 基础机制
- 玩家可将捕获的精灵"回顾"归还系统, 换取**洛克贝**.
- 每种精灵有基础回顾值 $v_0$ (种族值相关, 具体数值**待采集**).
- 回顾倍率取决于形态:

| 形态 | 倍率 (4.16 版本后) | 倍率 (之前) |
|---|---|---|
| 普通 | 1× | 1× |
| 污染 | **5×** | 3× |
| 炫彩 | **10×** | 3× |
| 异色 | **10×** | 3× |

### 2.2 经济意义
4.16 更新后, **每抓一只污染精灵就是一次 5× 倍率回顾**, 刷异色路径上每只污染体都是正收益而非沉没成本.
加上 **条件球不再消耗洛克贝** (从"1 咕噜 + 2 作物 + 洛克贝"改为"1 咕噜 + 1 作物"), 球的边际成本大幅下降.

## 3. 球的分类与成本

### 3.1 基础球
- **普通咕噜球 (潘多基础球)**: 最低级, 从任务/种植获得.
- **高级 / 超级咕噜球**: 更高基础捕捉率.
- **黄金咕噜球 / 新年球 / 恶魔球**: 稀有高捕率.
- **国王咕噜球**: 100% 捕捉.

### 3.2 条件球 (合成路径)
9 种条件球配方统一简化:
```
1 普通咕噜球 + 1 作物  →  1 条件球   (无洛克贝消耗)
```
其中:
- **网兜球**: 对 水/翼 系
- **好战球**: 对 火/龙 系
- **光合球**: 对 草/冰 系
- **瞌睡球**: 对睡眠状态
- **美妙球**: 对 萌/普通 系
- **调温球 / 补光球**: 另有属性适配
- 其他 (3 种) 具体待查实

### 3.3 炫彩球 / 棱镜球
- **100% 捕捉**, 直接赋予**炫彩形态** + **满资质** + **满个体值**.
- 获取极稀有 (战令 / 活动 / 商店限购), 单价高.
- 对"异色 → 异色炫彩"二次捕捉是关键期权.

## 4. 种族值与个体/努力值

### 4.1 六维种族值
精力 (HP) / 物攻 / 魔攻 / 物防 / 魔抗 / 速度. 每物种总和固定唯一.

### 4.2 计算公式
精力:
$$
\text{HP} = \frac{(2 \cdot B + I + E/4) \cdot L}{100} + L + 10
$$
其他五项:
$$
\text{Stat} = \left[ \frac{(2 \cdot B + I + E/4) \cdot L}{100} + 5 \right] \cdot N
$$
其中 $B$=种族值, $I$=个体值 (0-31), $E$=努力值 (0-255, 总 510 上限), $L$=等级, $N$=性格修正.

### 4.3 PVP 拉平
PVP 下等级拉平到固定值, 种族值不可变, **个体值和性格**成胜负关键 — 因此"6V" (六项 31 个体值) 和异色/炫彩的完美个体值加成有直接战略价值.

## 5. 属性克制

### 5.1 18 种属性
火/水/电/草/冰/武术/毒/土/翼/萌/虫/石/寂/龙/魔/机械/光 (另有第 18: 幽/恶/超 等变体, 版本间略有差异).

### 5.2 伤害系数
| 关系 | 系数 |
|---|---|
| 双重克制 | 3× |
| 克制 | 2× |
| 正常 | 1× |
| 抵抗 | 0.5× |
| 双重抵抗 | 1/3 |

## 6. 繁育系统

- **精灵蛋 5 种**: 完美蛋 / 剧情蛋 / 自选炫彩蛋 / 炫彩蛋 / 异色蛋.
- 同物种异性同蛋组 → "精灵远行" → 产蛋.
- **遗传**: 性格 / 个体值 / 部分技能有概率从父母继承.
- 父母若为炫彩/异色, 子代有**小概率**出炫彩蛋/异色蛋/炫彩异色蛋.
