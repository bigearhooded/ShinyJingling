# 08 · 经济参数（4.16 版本，2026 年 4 月）

> 用户提供 + 网络爬取交叉验证（B 站攻略 / 游民星空 / 3DM / 4399 / 腾讯新闻）

## 1. 球价（洛克贝-equiv）

| 球 | 价格 | 捕捉率 $q_\text{cap}$ | 来源 / 备注 |
|---|---|---|---|
| 普通咕噜球 | **0** (免费, 战斗/采集) | ~0.30–0.40 (估) | 攻略默认 baseline |
| 属性球（采花/挖矿） | **0** (时间成本) | ~0.85–0.95 (对应属性) | 用户提供 |
| 高级球 | **15,000** | ~0.65–0.80 (估) | 安妮商店, 多源确认 |
| 调温球 / 美妙球 (属性专属) | **7,500** | ~0.95 | 远行商人, 每日限购 100 |
| 捕光球 | **80,000** | **1.00** | S1 赛季大厅 (用户值 = 8w 完全吻合) |
| 国王球 | (商店) / 免费扭蛋 | **1.00** | 远行商人偶发 |
| 棱镜球 | **160 RMB** ≈ **3,200,000 洛克贝** | n/a (升格用) | 远行商人 4h 窗口 / 通行证 3 个/赛季 |

**汇率反推**: 棱镜球 160 RMB ≈ 320万洛克贝 ⇒ **1 RMB ≈ 20,000 洛克贝**

## 2. 保底机制

- 击破 **80 次噩梦污染** 必出异色 (确认硬保底, 与 Stage 2 LOO 选 M1 一致)
- 近期传闻官方下调至 70 次 (作为 sensitivity 分析)
- 每个家族池独立计数 (跨池切换会清零 — 即 V2 §3 cross_pool 数据的来源)

## 3. 概率

- 官方公告: **综合概率 ≈ 0.02** (per-击破)
- 玩家反推: per-trial $p_0 \approx 0.018$ (将 0.02 综合扣除保底贡献后反推)
- **本研究后验**: $p_0 \in [0.06, 0.09]$ (带 posting bias 校正), 即真实综合出货率 $\approx$ 0.06–0.09, **是官方 3–4.5 倍**

## 4. 收益

- $m_\text{poll} = 5$: 每只污染体回顾倍率 (相比普通精灵)
- $m_\text{shiny} = 10$: 异色回顾倍率
- $v_0$ 因精灵不同显著差异
  - 估计区间: 1,000–10,000 洛克贝/只 (普通)
  - 异色稀有精灵单价更高
  - **后续仿真使用 $v_0 = 5,000$ 作为典型值, 并对 [1000, 10000] 做 sensitivity**

## 5. 时间预算 (替代成本)

- 巡逻基础: 250 洛克贝/天
- 活跃玩家 (15-20 min/天): 1,000-3,000 洛克贝/天
- **棱镜球纯洛克贝兑换 ≈ 1,000+ 天 (3 年!)** → 实际只能氪金 / 通行证 / 远行商人偶遇

## 6. 星光值机制

- 不影响 per-trial $p_k$ 的异色概率
- 影响**污染产出速度** (每抓 N 只精灵触发 1 个污染)
  - 低阶宠物 (蘑菇, 蝌蚪): 1 球 1 个, 每只 ~1.5–2% 噩梦进度 ⇒ ~50-67 球出 1 污染
  - 一只污染体需 ~50 球, 80 污染保底 ⇒ ~4000 球次 (大量时间)
- 高星光值精灵: 攒污染快; 低星光值精灵: 慢
- 在仿真层: 体现为 **每小时可完成的"循环"次数** $\lambda$, 不进 $p_k$ 模型

## 7. 关键反转

**之前 Stage 4 使用 $v_0=50, c_\text{ball}=5$ 是数量级错误**, 用真实参数重新评估:

| 策略 (用 $v_0=5000$, $p_0=0.062$, M1) | 球选择 | 单循环净利润 (洛克贝) |
|---|---|---|
| S-P (纯保底, 80 次不抓) | 80 球过场 + 80000 捕光球抓异色 | $50000 - 80000 = $ **-30,000** ❌ |
| S-F + 捕光球 (全抓) | 80 × 80000 + 80000 = 6.48M | $80·5·5000 + 50000 - 6.48M = $ **-4.43M** ❌ |
| S-F + 高级球 (q=0.7) | 80 × 15000 = 1.2M | $80·0.7·25000 + 50000 - 1.2M - 80000 = $ **+170,000** ✅ |
| S-F + 普通球 (q=0.4, 免费) | 80 × 0 + 80000 异色球 | $80·0.4·25000 + 50000 - 80000 = $ **+770,000** ✅✅ |
| S-F + 属性球 (q=0.95, 免费) | 0 + 80000 | $80·0.95·25000 + 50000 - 80000 = $ **+1,870,000** ✅✅✅ |

**结论反转**: S-F 仍然最优, 但**球选择**比"是否全抓"更重要. 用免费属性球 (高 q) 比用付费高级球收益高 11 倍.

更要命的是: **如果 $v_0 < 1500$, 即使免费属性球也不赚** — 异色的洛克贝价值 < 80000 捕光球. 此时只能靠**收藏 utility** 或**升格炫彩**才回本.
